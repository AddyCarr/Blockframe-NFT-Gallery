# Blockframe

View and mint NFT's
Eaily share your gallery by loading your address!

## Installation

```bash
npm install
```

### Rename '.env.example.local' to '.env.local'and add the values

```
// .env.local
MORALIS_API_KEY=key_goes_here
BASE_URL=http://localhost:3000
```

## Run

```bash
npm run dev
```

## Using Prettier

```bash
npx prettier --write .
```
